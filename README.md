# AzureOne-Hub-x-ID
Prototipe ekosistem digital terpadu berbasis Azure untuk integrasi data industri dan inklusi ekonomi nasional
## 🚀 AzureOne Hub x ID - Digital Core Dashboard
This prototype demonstrates the integration of 26 industrial points into a unified Azure ecosystem.

### How to run:
1. Clone this repo
2. Run `npm install`
3. Run `npm run dev`
